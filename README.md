## memo

- LINE でリマインド
- share 機能で簡単に LINE 連携
- Google ログイン

- タグでタスクをフィルタ
- タスクの新規追加はモーダル
- タグの新規追加はモーダル
- 一度作ったタグは更新できない
- 既存のタスクをタップで編集モーダル(期限、タイトル、詳細を変更できる)
- 既存のタスクのタグアイコンアップでタグ編集モーダル
